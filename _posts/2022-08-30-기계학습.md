<h1>Numpy</h1>

```
import numpy as np

x = np.arrange(0, 5)
```

Numpy는 "Numerical Python"의 약자이다.  

고전적인 배열 선언도, 리스트 컴프리헨션도 없이 Numpy 만으로 아주 간단히 배열을 생성할 수 있다.

<h3>Numpy 배열의 속성</h3>
- ndim : 차원(= 축의 개수), Rank라고도 함  
예를 들어, 3차원의 ndim은 3임
- shape : 각 차원의 배열의 크기  
예를 들어, 2x3 배열의 Shape은 (2, 3)임.
- size : 배열의 모든 원소의 개수
- dtype : 원소의 자료 형식
