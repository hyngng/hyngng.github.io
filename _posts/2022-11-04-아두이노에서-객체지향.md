```cpp
#include "Arduino.h"
#include "MainFunctions.h"

int MainFunctions::AddFiftyWon(int Money)
{
  Money += 500;
  return Money;
}

{
  Money += 1000;
  return Money;
}
```

# 타 언어와 C++의 차이점  

- C++는 클래스(.cpp) 안에서 변수와 함수를 정의한다.
- C++는 헤더(.h) 파일에서 변수와 함수에 대한 세부사항을 따로 작성한다.
