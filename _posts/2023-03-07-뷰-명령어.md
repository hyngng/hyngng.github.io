<br>

- Vue Cli 설치  

```bash
npm install -g @vue/cli
```

<br>

- Vue 프로젝트 생성  

```bash
vue create >project_name_here<
```

<br>

- Vue 프로젝트 빌드

```bash
npm run serve
```

  - 빌드 중지는 Ctrl + C 단축키로 해결할 수 있다.
