<h6>기기와의 연결 확인</h6>
```
adb devices
```
  

---


<h6>기기내 어플 제거 명령어</h6>
```
adb shell pm uninstall -k --user 0 "pkg_name"
```

<h6>기기내 어플 비활성화 명령어</h6>
```
adb shell pm disable --user 0 pkg_name
```

<h6>상단바 숨기기</h6>
```
adb shell settings put global policy_control immersive.status=*
```
  

---


<h6>기기와의 연결 끊기</h6>
```
adb kill-servers
```
