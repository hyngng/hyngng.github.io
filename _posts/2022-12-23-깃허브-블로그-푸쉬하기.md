블로그에 오래 소홀히 하다보면 간단한 명령어도 까먹는 일이 생겨서, 몇 개 기본 명령어를 메모해둔다.  
커맨드 창에 다음의 명령어를 입력하여 푸쉬한다.  

```
cd kiw6024.github.io
git add .
git commit -m "이름"
git push
```
